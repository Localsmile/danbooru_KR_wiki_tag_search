프롬프트 변환기

NAI ↔ 로컬(A1111/Forge) 프롬프트 문법 상호 변환.

방향

  • NAI → Local : NAI 프롬프트를 로컬 형식으로 변환
  • Local → NAI : 로컬 프롬프트를 NAI 형식으로 변환

NAI 문법

  • {tag} 강조 ×1.05 / [tag] 약화 ÷1.05 (겹칠수록 곱셈)
  • 1.5::tag :: 수치 강조 (V4+)
  • -1::tag :: 음수 가중치 (V4.5+)
  • artist:name 아티스트 태그

로컬 문법

  • (tag:1.2) 가중치 지정
  • \(tag\) 괄호 이스케이프
  • @(name) 아티스트 태그 (anima 모델 한정)

아티스트 처리

  • NAI → Local : artist: 제거 후 이름만 또는 @()로 출력 (anima용)
  • Local → NAI : CSV DB의 아티스트에 자동 artist: 부착

가중치 정규화 (NAI → Local)

NAI는 가중치 높아도 잘 버티는데 로컬은 그대로 넣으면 그림이 깨질 수 있음. 정규화로 로컬에 맞게 줄여줌.
  • 끔 : 가중치 그대로 변환
  • 고정 압축 : 모든 가중치를 일정 비율로 줄임. 압축률 낮을수록 더 세게 줄이고, 최대값 넘으면 잘라냄
  • 상대값 : 제일 높은 가중치를 목표값까지 내리고 나머지도 같은 비율로 따라감. 태그끼리 강약 비율은 유지
