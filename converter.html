용도

NAI와 로컬/PixAI 계열 프롬프트 표기 상호 변환.

기본

  • NAI → Local : {tag}, 1.2::tag ::, artist:name → 로컬용 표기
  • Local → NAI : (tag:1.2), @(artist) → NAI용 표기
  • 아티스트 이름 자동 인식. 방향에 따라 artist: 또는 @() 적용

가중치 정규화

NAI 가중치를 로컬/PixAI (SDXL)에서 사용 시 문제가 발생할 가능성이 높기에 가중치 정규화 시도.
  • 원본 — 숫자 유지. 비교용
  • 최댓값 맞춤 — 가장 강한 태그를 지정한 최댓값에 맞추고 전체 비율 동시 축소
  • 비례 축소 — 가중치 변동 폭 축소. 강도 50%면 1.4 → 1.2

아티스트

  • 이름만 : artist:name → name
  • @() 변환 : artist:name → @(name)

주의

  • 마이너스 가중치는 별도 칸에 분리
  • 괄호 포함 아티스트는 \(, \) 유지 처리
